Welcome to Your AI Development Environment

This server is running the iTaK AI Stack, a comprehensive collection of tools for AI development, automation, and data management.

⚠️ Security Notice

For security reasons, the full dashboard is only accessible from localhost (on the server itself). This public page provides information about the available services without exposing sensitive controls.

How to Access the Full Dashboard

  1. Connect directly to the server (via SSH, RDP, or physical access)
  2. Open a web browser on the server
  3. Navigate to http://localhost
  4. You'll see the full dashboard with all controls and services

Available Services

n8n

Workflow automation platform for connecting various services and APIs.

Use cases: Create automated workflows, connect to APIs, process data

OpenWebUI

Web interface for interacting with Ollama's large language models.

Use cases: Chat with AI models, generate text, create AI assistants

Ollama

Run large language models locally with a simple API.

Use cases: Local AI inference, custom model deployment

SearXNG

Privacy-respecting metasearch engine.

Use cases: Private web searches, research, data gathering

YouTube Downloader

Tool for downloading videos from YouTube and other platforms.

Use cases: Content archiving, offline viewing, data collection

Web Scraper (Crawlee)

Modern web scraping API powered by Playwright and Crawlee.

Use cases: Data extraction, web automation, content monitoring

Flowise

Build LLM apps with a visual interface.

Use cases: Create AI workflows, build chatbots, design agents

Portainer

Container management UI for Docker.

Use cases: Manage containers, monitor resources, deploy applications

Adminer

Database management tool.

Use cases: Manage databases, run queries, import/export data

Supabase (PostgreSQL)

PostgreSQL database for application data storage.

Use cases: Data storage, application backend, structured data

Integration Examples

Here are some ways you can combine these services:

  • AI-Powered Data Pipeline: Use the Web Scraper to collect data, process it with n8n, store it in Supabase, and analyze it with Ollama.
  • Content Creation Workflow: Research topics with SearXNG, download reference videos with YouTube Downloader, and generate content with OpenWebUI.
  • Custom AI Agent: Build an agent in Flowise that uses Ollama for reasoning, n8n for actions, and Web Scraper for gathering information.
